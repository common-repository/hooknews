== Description ==
Submit your content to hooknews.com by API.


== Installation ==

1. Copy the entire /hooknews/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. Click the "Settings" admin menu link, and select "HookNews".
4. Configure your account and default categories.
